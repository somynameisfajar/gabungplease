# gabungplease
Convenient Data Consolidation Utility

Gabungplease is a lightweight, secure, and fully offline desktop utility designed to streamline the consolidation of tabular data. It combines powerful data processing with a clean, modern interface, allowing users to merge and manage their datasets without compromising privacy.

Key Features :

100% Offline Processing: Total data sovereignty. Gabungplease runs entirely on your local machine and never sends data to external servers, making it safe for sensitive or confidential information.

Robust Tabular Data Engine: Powered by pandas under the hood to handle, clean, and merge complex tabular data quickly and accurately. It features multi-thread data processing to deliver the result faster (>30% faster than alpha version with just single-threading)

Modern GUI: Features a sleek, dark-mode-friendly graphical interface built with customtkinter for a frictionless user experience.

Standalone Executable: Compiled natively using Nuitka. You don't need to install Python, dependencies, or set up virtual environments—just download the .exe and run it.
