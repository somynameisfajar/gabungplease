What it does

Tedious task, made easy.
Nothing you don't.

Built for many of us, Less clicks, Less stress at your task.

📂
Multi-format support

Reads .xlsx, .xls, .xlsb, and .csv in the same run. Detects delimiters automatically — no config needed.

🔀
Parallel file reading

Processes up to 6 files simultaneously. SSD Mode cranks up threads for even faster throughput on fast storage.

🧹
Deep XML cleaning

Strips illegal XML characters and hidden line breaks that cause Excel's "Repair" prompt — silently, automatically.

📅
Date standardisation

Detects date columns by keyword and normalises them to YYYY-MM-DD — consistent across every source file.

🔍
Duplicate removal

Drops exact duplicate rows across all files after merge, with an accurate count reported at the end.

📋
Schema alignment log

When files have different column structures, Gabungplease auto-aligns them and writes a clear diff log to disk.

Workflow

Three steps.
One clean file.

01

Select your folder

Point Gabungplease at any folder containing your Excel or CSV files. Use the keyword filter to process only files whose names match a specific term — useful when a folder contains multiple report types.

02

Configure and consolidate

Toggle deep cleaning, date standardisation, and duplicate removal. Click the big button. Gabungplease reads all matching files in parallel, aligns their columns automatically, and merges everything into one DataFrame.

03

Save as .xlsx or .csv

A save dialog appears. If you don't respond within 30 seconds, it auto-saves to the source folder so nothing is lost. A summary shows total files, rows, and duplicates removed.
